# Sum of Numbers

## Problem Description

You are given a list of integers, one per line. Your task is to calculate the **sum** of all the numbers.

## Input Format

- The input consists of multiple lines
- Each line contains a single integer
- The input may have empty lines (which should be ignored)

## Output Format

- Return a single integer: the sum of all numbers

## Example

**Input:**
```
1
2
3
4
5
```

**Output:**
```
15
```

## Explanation

The sum is: 1 + 2 + 3 + 4 + 5 = 15

## Notes

- Numbers can be negative
- Empty lines should be ignored
- The result should be returned as a string representation of the integer
